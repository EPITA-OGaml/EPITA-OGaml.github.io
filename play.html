Instructions d'installation
Windows

Avec l'installeur

  1. Télécharger l'installeur
  2. Double-cliquer sur l'installeur
  3. Choisir si le jeu doit-être installé seulement sur la session active (recommandé), ou pour tous les utilisateurs (nécessite les droits administratifs)
  4. Suivre les instructions de l'installeur
  5. Ouvrir le menu Démarrer, chercher le dossier "Purest Atom Exam"
  6. Pour lancer le jeu, cliquer sur le logiciel "Purest Atom Exam"

Avec l'archive zip

  1. Télécharger l'archive zip
  2. Faire clic droit sur l'archive
  3. Sélectionner "Extraire tout"
  4. Suivre les instructions
  5. Ouvrir le dossier où le jeu a été extrait
  6. Double-cliquer sur le fichier Purest Atom Exam.exe
Linux
  1. Télécharger l'archive zip
  2. Décompresser l'archive
  3. Se rendre dans le dossier où le jeu a été extrait
  4. Faire clic droit sur le fichier Purest Atom Exam.x86_64
  5. Dans l'onglet "Permissions", activer l'option "Autoriser l’exécution du fichier comme un programme"
  6. Fermer les options
  7. Double cliquer sur le fichier Purest Atom Exam.x86_64
Mac OS X
  1. Télécharger l'archive zip
  2. Décompresser l'archive
  3. Se rendre dans le dossier où le jeu a été extrait
  4. Double cliquer sur l'exécutable
Désinstaller le jeu
Windows

Depuis le menu Démarrer (Windows 7)

  1. Ouvrir le menu Démarrer
  2. Cliquer sur Tous les programmes
  3. Chercher le dossier Purest Atom Exam
  4. Cliquer sur "Désinstaller Purest Atom Exam"

Depuis le menu Démarrer (Windows 10)

  1. Chercher le dossier Purest Atom Exam
  2. Faire clic droit sur le logiciel puis Désinstaller
  3. Chercher Purest Atom Exam dans la liste des programme
  4. Faire clic droit sur le programme puis "Désinstaller"

Depuis le Panneau de configuration

  1. Sous Programmes, cliquer sur Désinstaller un programme
  2. Chercher Purest Atom Exam version x dans la liste des programmes (le x dépend de la version installé, exemple : 1.0)
  3. Faire clic droit sur le programme puis Désinstaller

Depuis le dossier de Purest Atom Exam

  1. Double-cliquer sur le fichier unins000.exe
Linux
  1. Supprimer le dossier Purest Atom Exam
Mac OS X
  1. Supprimer le dossier Purest Atom Exam
